Парсит чаты по запросам, что формируются из файлов citys.txt и words.txt. Запускает парс последовательно на каждом аккаунте. Аккаунты вместе должны находиться в папке sessions. Поддерживается формат аккаунтов session+json. Результат парса записывается в файлы внутри папки results.
