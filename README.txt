Парсит чаты по запросам, что формируются из файлов citys.txt и words.txt. Запускает парс последовательно на каждом аккаунте с рассчетом, что на 1000 запросов (количество городов х количество слов) необходим 1 аккаунт. Аккаунты вместе должны находиться в папке sessions. Результат парса записывается в файлы внутри папки results.
